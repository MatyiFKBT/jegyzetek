## FileSystem

```java
FileSystem fs;
try {
	fs = FileSystem.get(conf);
	if (fs.exists(new Path("tempOraiOut")))
		fs.delete(new Path("tempOraiOut"),true);
} catch (IOException e1) {
	e1.printStackTrace();
}
```
